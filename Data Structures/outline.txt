Course Contents Course Code: CSC211 Course Title: Data Structures and Algorithms Credit Hours: 4(3,1) Corequisite(s): Prerequisite(s): CSC103 
Course Learning Outcomes / Course Objectives : CLO's for Theory CLO-1 Employ linear data structures to solve computing problems. Applying CLO-2 Use non-linear data structures to solve computing problems. Applying CLO-3 Analyze the time complexity of various algorithms. Analyzing CLO's for Lab CLO-4 Implement data structures and algorithms. Applying CLO-5 Develop a project using appropriate data structures in a team environment. Creating 
Syllabus / Theory Course Outline : Unit Topics 1. Data Structure: Overview, Importance, Classification, Operations, and Abstract Data Types. 2. Static List; Dynamic List: Single Linked Lists, Circular Linked List, Doubly Linked List, and Applications. 3. Stack: Concept, and Applications. 4. Queue: Concept, Types and Applications. 5. Tree: Concept & Terminology, Traversal Algorithms; Binary Search Trees & its Various Operations; AVL Tree and Min & Max Heaps. 6. Graph: Basic Concepts & Terminology; Representation, Types; Graph Traversal Techniques: Breadth First Search, Depth First Search; Minimum Spanning Trees: Kruskal Algorithm, Prims Algorithm; and Shortest Path Problem: Dijkstra's Algorithm. 7. Sorting algorithms: Bubble, Selection, Insertion & Merge Sort, Searching Algorithms: Linear & Binary Search; Hashing: Hash Functions, Hash Tables, Strategies for Avoiding & Resolving Collisions. 8. Complexity Analysis: Growth Rate of Function; Asymptotic Notation; Time Complexity of Searching and Sorting Algorithms. 
Practical Requirements : Fundamentals of C++; Static List(Array Based), Singly Linked List, Doubly Linked list, Circular Linked list, Stack; Queue; Binary Search Tree, AVL Tree, Heap Tree as Priority Queue; Graph representation by Matrix, Graph traversals, Dijkastra Algorithm; Minimum Spanning Tree; Searching and Hashing Algorithms; Sorting Algorithms 
Suggested Instructional Reading Material : Textbook: 1. A Common-Sense Guide to Data Structures and Algorithms, Jay Wengrow, Pragmatic Bookshelf, 2020. Reference Book: 1. Data Structures and Algorithm Analysis in C++, Mark Allen Weiss, Addison-Wesley, 2014.
